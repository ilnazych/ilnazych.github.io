# projects

Здравствуйте. 
Для запуска локально вы можете скачать используя команду "git clone https://github.com/ilnazych/ilnazych.github.io.git" в вашем редакторе кода.
После копирования необходимо с помощью команды git checkout необходимо перейти в ветку antd-table.
Далее установите npm используя команду npm i

Для запуска проекта используйте команду npm run dev

Спасибо за внимание.
